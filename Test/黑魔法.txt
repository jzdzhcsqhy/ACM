#include <stdio.h>

int main(void)
{
    int x = 5;
    printf("%d and ", sizeof(x++)); // note 1
    printf("%d\n", x); // note 2
    return 0;
}

这一段的输出结果是  4 and 5

注意到，虽然执行了sizeof(x++),但是输出的时候，x的值还是没有变。
因为sizeof是编译时行为，在运行时就不执行了。
在汇编代码中可以看到，sizeof(x) 被直接编译成了立即数，并没有对x进行自增。也就是说在执行过程中，根本不会执行x++这条语句。
